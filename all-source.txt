================================================================================
  공통 게임 개요
================================================================================

- 장르: 2D 탑다운 웨이브 서바이벌 슈팅
- 엔진: Phaser 3.60 (CDN)
- 조작: WASD / 방향키 이동, 공격은 가장 가까운 적에게 자동 발사
- 일시정지: SPACE
- 씬 구성: MenuScene → GameScene → GameOverScene

난이도 3단계 (쉬움/보통/어려움):
  - 쉬움: 체력200, 적 느림, 스폰 적음
  - 보통: 체력100, 균형
  - 어려움: 체력80, 적 빠르고 많음

적 타입 4종:
  - fast (빨간 원): 빠르지만 약함, HP 낮음
  - tank (주황 사각): 느리지만 강함, HP 높음 (항상 2방에 죽도록 조정)
  - goldCircle (노란 원): fast의 희귀 변종, 죽이면 이동속도 버프 드롭
  - goldRect (노란 사각): tank의 희귀 변종, 죽이면 공속 버프 드롭

공통 시스템:
  - 웨이브: 30초마다 다음 웨이브, 적 HP/속도/스폰수 증가
  - 회복약: 모든 적 처치 시 초록 구슬 드롭 (HP +5)
  - 파워업: 노란 변종 처치 시 SPD+5% 또는 ATK.SPD+5% 드롭
  - 파워업 자석 흡수: 플레이어 근처(80px) 파워업이 자동 흡수
  - UI: 체력바, 웨이브, 타이머, 킬수, 버프현황 표시

================================================================================
  [MODE 2: Wide] — 넓은 화면 모드 (카메라 추적)
================================================================================

특징:
  - 월드 2048×1536 (뷰포트의 2배), 카메라가 플레이어 부드럽게 추적
  - 카메라: lerp 0.04, 넓은 데드존(55%), 라운드픽셀
  - 맵 경계: 두꺼운 빨간 테두리 + 바깥 어둡게 처리

Wide 전용 시스템:

  1) 폭탄 시스템
     - 자동 폭탄: 100발째 탄환이 삼각형 폭탄으로 발사 (2×2 격자 범위, 접촉 즉시 폭발)
     - 수동 폭탄 (R키): 보유 폭탄 소모, 3×3 격자 범위, 적에 닿으면 설치 → 3초 후 폭발
     - 폭발 지속: 0.5초간 범위 내 진입하는 적도 모두 처치
     - 폭발 이펙트: 격자 단위 주황색 하이라이트

  2) 자폭형 적 (빨간원 100마다 1체)
     - 옅은 빨간 원, 약간 더 큼, 깜빡임
     - 10초 후 또는 처치 시 2×2 범위 폭발 (빨간 이펙트)
     - 자폭 처치 시 초록 회복약 드롭 없음

  3) 성장 시스템
     - 킬 100마다 최대 체력 +10%
     - 초록 회복약 100개 수집마다 폭탄 +1개 획득
     - 노란 변종 출현 확률: 기본 5% + 웨이브마다 2% 증가 (최대 30%)

  4) UI 추가
     - BOMBS: 보유 폭탄 수 (R키 표시)
     - HEAL: 회복약 수집 카운터 (100개마다 리셋)
     - 난이도 옆 [Wide] 표시
